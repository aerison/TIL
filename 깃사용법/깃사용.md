   ```console

   ```

   

2. github -> B(폴더)(clone)(no init-이미 master까지 클론되어옴)

   ```console
   #vagrant에서,
   $git clone <클론 할 레포주소> B 
   ```

   

3. B a.txt 수정, 커밋

4. B ->github

5. A a.txt 수정, 커밋

6. github -> A 

   ```console
   $git pull origin master
   ```

1. git branch[https://backlog.com/git-tutorial/kr/]

   ```console
   #브랜치만들기
   $git branch dev

   $git checkout dev

   #브랜치 만들고, 글까지 쓰기
   $ git checkout -b 반장님

   $git checkout master
   ```
