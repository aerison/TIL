vagrant up후 

1. A(폴더) a.txt 만들고 커밋 -> github

   ```console
   mkdir A
   ```

   

2. github -> B(폴더)(clone)(no init-이미 master까지 클론되어옴)

   ```console
   #vagrant에서,
   $git clone <클론 할 레포주소> B 
   ```

   

3. B a.txt 수정, 커밋

4. B ->github

5. A a.txt 수정, 커밋

6. github -> A 

   ```console
   $git pull origin master
   ```

7. A ->github  ***에러발생(아톰열어서, 선택)

   7-1.github -> A ***에러발생
